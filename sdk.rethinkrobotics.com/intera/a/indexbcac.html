Overview

The procedure below describes the process required to update the software on a Sawyer SDK Robot that is currently running in SDK mode.
Note: The instructions below are only related to the Sawyer robot.

Requirements

  • USB Keyboard
  • Intera Updater zip file containing the update .pkg and .sig files, unzipped
  • FAT 32 Formatted USB with Intera Updater .pkg and .sig saved at the Top Level
  • Sawyer Robot Running SDK 5.0.4. If you are upgrading an Intera 3.3.7 Sawyer, please see follow the Upgrade Sawyer to Intera SDK tutorial instead.

Procedure

1. Boot Into the Field Service Menu (FSM)

  1. Connect a USB keyboard into the robot controller.
  2. Shut down the robot by pressing the power button on the controller. After the robot has completely powered down (fan stopped, screen back light is off), wait for 5 seconds and press the power button again to reboot the robot.
  3. On the keyboard, start pressing CTRL+F continuously until the FSM menu appears on the robot screen.
    4. Tip: Look for the screen below when rebooting. When the text Preparing to boot Intera… appears on the upper left of the screen, pressing CTRL+F' will bring you right into the FSM.

  4. When successful, the Main Menu of the FSM will appear, see the screen below for reference.

2. Reboot Into Intera

  1. Using the arrow keys on the keyboard, navigate down to highlight the check box for “Next boot in: Intera”.
  2. When this selection is highlighted press Enter to check this box.
    3. Note: If the currently installed version of software is 5.0.4 and the robot is being updated to 5.1, a prompt requesting a password will appear. The password is "reywas17990". After entering the password, tab to "Yes" and press enter.

  3. The Main Menu will again appear on the screen. Using the arrow keys on the keyboard, go to and select "Reboot".
  4. Press "Yes" to confirm the reboot.

3. Update the Robot Using the Intera UI

  1. When the robot finished booting, press the Navigator Wheel to automatically home the robot.
  2. Insert the USB drive with the updater into an empty USB port on the controller.
  3. Press the Rethink Button on the Navigator to show the circle menu. See the image below for reference.

  4. Using the Navigator Wheel, scroll to and select System by pressing the Navigator Wheel.
  5. On the next menu, scroll to and select "Upgrade Software".

  6. From the list that appears, scroll to and select the version to which the robot will be updated.

  7. A confirmation window will appear, select "Update" to confirm and start the software upgrade.
  8. The update process will start and the robot will automatically reboot during this time. Once the update is complete and the robot has rebooted, home the robot automatically to open Intera.

4. Switch Back to SDK Mode

  1. Shut down the robot and boot into the FSM using the instructions detailed in Step 1.
  2. Once in the FSM, use the arrow keys on the keyboard to select "Intera-SDK". See the image below for reference.

  3. A prompt will appear asking the user to read the General Disclaimer. After reading through the disclaimer, tab to and select "Yes".

  4. Move to and select "Reboot".
  5. The robot will shut down and boot directly into the Intera-SDK.
